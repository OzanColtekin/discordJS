# discordJS

### 19 Police Pursuit

İlk olarak SAMP platformu üzerinde açılan sunucunun topluluk sunucusu olan discord sunucusunda kullanılmak üzere tasarlanmıştır. <br>
Ardından kişisel discord botu olarak düzenlenmiştir. <br>
Kullanmak için `index.js` üzerinde değişiklikler yapılması gerekmektedir, her sunucunun kendi rol idleri eklenmeli, komutlar dosyası içerisinde olan kontroller değiştirilmelidir.<br>
Bota yeni komut eklenmek istediğinde komutlar dosyası içerisinde `komutadı.js` şeklinde dosya oluşturulup içerisinde kodlar yazılabilir.
